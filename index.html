PANDUAN APLIKASI

 

1.       Halaman Depan

Dari halaman ini, terdapat tombol untuk memulai diagnose penyakit Ikterus pada bayi, dengan metode Fuzzy Logic Mamdani

 

 

Bila diklik tombol menu sebelah kiri atas akan muncul menu untuk mengakses seluruh fungsi aplikasi

 

 

2.       Diagnosa Penyakit

Diagnosa penyakit di mulai dengan menjawab pertanyaan dari 1-7

 

 

 

Bila pertanyaan terakhir sudah di jawab akan muncul hasil dari inferensi fuzzy logic menunjukkan deteksi penyakit Ikterus pada bayi

 

 

Diagnosa bisa dijalankan kembali dengan menekan tombol ULANGI DIAGNOSA

 

Untuk melihat langkah penanganan terhadap salah satu penyakit, bisa di klik link Penanganan Ikterus Fisiologis atau  Penanganan Ikterus Patologis. Bila diklik akan muncul halaman langkah-langkah yang bias diambil untuk penyakit tersebut

 

 

3.       Login

Login digunakan apabila admin ingin mengubah FLL dan Pustaka yang digunakan oleh aplikasi

 

 

 

4.       Fuzzy Logic Controller

Fuzzy Logic Controller berisi FLL (Fuzzy Logic Language) yang digunakan oleh aplikasi dalam memproses gejala penyakit Ikterus pada bayi

 

 

5.       Pustaka

Pustaka berisi berbagai istilah dan keterangan lain untuk mendukung informasi yang digunakan oleh aplikasi
